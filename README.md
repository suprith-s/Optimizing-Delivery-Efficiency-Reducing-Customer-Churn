I Recently Participated in "The Data Decode Challenge" by the Indian Institute of Science (IISc), Bangalore, in March 2026
It happened online, and it was a single intensive round, where the challenge was to clean the raw dataset, find insights,
And present them on why there was a reduction in customer churn.
So, this repository is all about the work I did for the Data Decode Challenge.

I used 2 tools for analyzing and visualization
> EXCEL
> POWER BI

First, I cleaned the dataset using Excel and also created some new data columns (from the given data itself) 
and then I uploaded the dataset to Power BI, and made some changes in Power Query, and created some quick measures(DAX)
and then created a visualization dashboard(& charts) and found out the problems/issues in reducing of customer churn.
and presented the major reasons for customer churn and some strategies(suggestions) to overcome that.
